PORTFOLIO

Weather App

GitHub || Web

Project Duration - 2 days

Stack: NodeJS, Express Server, DarkSky API, Mapbox API, HTML, CSS

This is a web application built with NodeJS on Express Server using two common APIs, DarkSky and Mapbox, and a responsive front-end. The app takes in user input for city, translates the text to coordinates using Mapbox and uses those to fetch weather data from DarkSky. Deployed on Heroku and built in ES6.
